# 4.3.9.5. OLE 개체

Tag ID: HWPTAG_SHAPE_COMPONENT_OLE

**표 117: OLE 개체** — 개체 공통(표 68) + OLE 개체 속성(표 118) 24바이트.

**표 118: OLE 개체 속성**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| UINT16 | 2 | 속성(표 119) |
| INT32 | 4 | extent x |
| INT32 | 4 | extent y |
| UINT16 | 2 | BinData ID |
| COLORREF | 4 | 테두리 색 |
| INT32 | 4 | 테두리 두께 |
| UINT32 | 4 | 테두리 속성(표 87) |
| 전체 길이 | 24 | |

**표 119: OLE 개체 속성의 속성** — bit 0-7 drawing aspect (DVASPECT_CONTENT=1, THUMBNAIL=2, ICON=4, DOCPRINT=8), bit 8 TRUE if moniker is assigned, bit 9-15 베이스라인(0 기본 85%, 1-101은 0-100%), bit 16-21 개체 종류(0 Unknown, 1 Embedded, 2 Link, 3 Static, 4 Equation).

---

## 실제 구현 시 주의사항 (Implementation Notes)

표 119에서 **bit 16-21**은 "개체 종류"를 나타낸다고 명시되어 있으나, 실제로 속성 필드는 **UINT16 (16비트)** 타입이다. UINT16은 16비트이므로 **bit 16-21은 이 필드에 존재하지 않는다.** 실제 구현에서는:

- bit 0-7: drawing aspect
- bit 8: has moniker
- bit 9-15: baseline

만약 "개체 종류" 정보가 필요하다면, 이후 데이터 필드에서 별도로 읽어야 할 수 있다.
