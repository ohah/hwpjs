**표 34: 글꼴에 대한 언어** — 0: 한글, 1: 영어, 2: 한자, 3: 일어, 4: 기타, 5: 기호, 6: 사용자

**표 35: 글자 모양 속성** — bit 0: 기울임, bit 1: 진하게, bit 2-3: 밑줄 종류, bit 4-7: 밑줄 모양, bit 8-10: 외곽선 종류, bit 11-12: 그림자 종류, bit 13: 양각, bit 14: 음각, bit 15: 위 첨자, bit 16: 아래 첨자, bit 17: Reserved, bit 18-20: 취소선/강조점, bit 21-24: 강조점 종류, bit 25: 글꼴에 어울리는 빈칸, bit 26-29: 취소선 모양(표 25 참조), bit 30: Kerning
