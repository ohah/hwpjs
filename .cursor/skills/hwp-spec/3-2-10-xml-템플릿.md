# 3.2.10. XML 템플릿

`XMLTemplate` 스토리지에는 XML Template 정보를 저장한다.

`_SchemaName` 스트림에는 Schema 이름 문자열이 저장된다.

**표 10: Schema 이름 정보**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| DWORD | 4 | Schema 이름 길이 (len1) |
| WCHAR array[len1] | 2×len1 | Schema 이름 |
| 전체 길이 | 4 + (2×len1) | |

`Schema` 스트림에는 Schema 문자열이 저장된다.

**표 11: Schema 길이 정보**
