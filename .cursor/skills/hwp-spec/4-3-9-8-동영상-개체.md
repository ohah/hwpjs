# 4.3.9.8. 동영상 개체

Tag ID: HWPTAG_VIDEO_DATA

**표 122: 동영상 개체** — 개체 공통(표 68) + 동영상 개체 속성(표 123).

**표 123: 동영상 개체 속성**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| INT32 | 4 | 동영상 타입(표 124) |
| BYTE stream | n | 타입별(표 125, 126) |
| 전체 길이 | 가변 | 4+n |

**표 124: 동영상 타입** — 0 로컬 동영상, 1 웹 동영상.

**표 125: 로컬 동영상 속성** — UINT16 비디오 BinData ID, UINT16 썸네일 BinData ID. 전체 4.

**표 126: 웹 동영상 속성** — WCHAR array[len] 웹 태그, UINT16 썸네일 BinData ID. 전체 가변 (2×len)+2.
