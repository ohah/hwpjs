# 4.3.9.7. 묶음 개체

Tag ID: HWPTAG_SHAPE_COMPONENT_CONTAINER

**구현 상태**: 구현 완료. 테스트 파일(noori.hwp)에 SHAPE_COMPONENT_CONTAINER 레코드가 없어 실제 파일로 테스트되지 않음.

---

**표 120: 묶음 개체**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| BYTE stream | n | 개체 공통 속성(표 68) |
| BYTE stream | n2 | 묶음 개체 속성(표 121) |
| BYTE stream | n3 | 개체 속성 × 묶음 개체 수 (그리기, OLE, 그림, 묶음) |
| 전체 길이 | 가변 | n+n2+n3 |

**표 121: 묶음 개체 속성**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| WORD | 2 | 개체 개수(n) |
| UINT32 array[n] | 4×n | 개체 컨트롤 ID 배열 |
| 전체 길이 | 가변 | 2+(4×n) |
