# 4.3.10.9. 쪽 번호 위치

Tag ID: MAKE_4CHID('p','g','n','o')

**표 147: 쪽 번호 위치**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| UINT32 | 4 | 속성(표 148) |
| WCHAR | 2 | 사용자 기호 |
| WCHAR | 2 | 앞 장식 문자 |
| WCHAR | 2 | 뒤 장식 문자 |
| WCHAR | 2 | 항상 "-" |
| 전체 길이 | 12 | |

**표 148: 쪽 번호 위치 속성** — bit 0-7 번호 모양(표 143 참조), bit 8-11 표시 위치: 0 없음, 1-6 좌상/중상/우상/좌하/중하/우하, 7-10 바깥쪽 위/아래, 안쪽 위/아래.
