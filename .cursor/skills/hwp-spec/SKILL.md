# HWP 5.0 스펙 (소제목별)

한글 문서 파일 형식 5.0 명세를 **소제목 단위**로 나눈 내용이다. 원문 링크 없이 각 파일에 스펙 본문(표·설명)을 직접 넣어 두었다.

## 사용 방법

- 구현·파싱·버그 수정 시 **해당 소제목과 같은 번호의 .md 파일**을 열어 참조한다.
- 예: FileHeader 파싱 → `3-2-1-파일-인식-정보.md`, 문서 속성 → `4-2-1-문서-속성.md`, 표 개체 → `4-3-9-1-표-개체.md`.
