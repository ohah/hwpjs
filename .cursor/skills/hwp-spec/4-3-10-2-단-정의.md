# 4.3.10.2. 단 정의

Tag ID: MAKE_4CHID('c','o','l','d')

**표 138: 단 정의**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| UINT16 | 2 | 속성 bit 0-15(표 139) |
| HWPUNIT16 | 2 | 단 사이 간격 |
| WORD array[cnt] | 2×cnt | 단 너비(동일하지 않을 때 cnt개) |
| UINT16 | 2 | 속성 bit 16-31(표 139) |
| UINT8 | 1 | 단 구분선 종류 |
| UINT8 | 1 | 단 구분선 굵기 |
| COLORREF | 4 | 단 구분선 색상 |
| 전체 길이 | 가변 | 12+(2×cnt) |

**표 139: 단 정의 속성** — bit 0-1 단 종류(0 일반 다단, 1 배분, 2 평행), bit 2-9 단 개수 cnt(1-255), bit 10-11 단 방향(왼쪽/오른쪽/맞쪽), bit 12 단 너비 동일 여부.
