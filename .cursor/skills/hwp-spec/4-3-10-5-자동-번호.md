# 4.3.10.5. 자동 번호

Tag ID: MAKE_4CHID('a','u','t','n')

**표 142: 자동 번호**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| UINT32 | 4 | 속성(표 143) |
| UINT16 | 2 | 번호 |
| WCHAR | 2 | 사용자 기호 |
| WCHAR | 2 | 앞 장식 문자 |
| WCHAR | 2 | 뒤 장식 문자 |
| 전체 길이 | 12 | |

**표 143: 자동 번호 속성** — bit 0-3 번호 종류(0 쪽, 1 각주, 2 미주, 3 그림, 4 표, 5 수식), bit 4-11 번호 모양(표 134 참조), bit 12 superscript(각주 번호 위첨자 여부).
