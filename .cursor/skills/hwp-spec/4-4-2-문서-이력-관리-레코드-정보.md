# 4.4.2. 문서 이력 관리 레코드 정보

히스토리 아이템은 HISTORY_RECORD_TYPE_STAG 레코드로 시작하고, HISTORY_RECORD_TYPE_ETAG 레코드로 끝난다.

###### 4.4.2.1. 히스토리 아이템 정보 시작

Tag ID: HISTORY_RECORD_TYPE_STAG (0x10)

**표 154: 히스토리 아이템 정보 시작** — WORD flag, UINT option.

**flag**: HISTORY_INFO_FLAG_VERSION (0x01) Version, HISTORY_INFO_FLAG_DATE (0x02) Date, HISTORY_INFO_FLAG_WRITER (0x04) Writer, HISTORY_INFO_FLAG_DESCRIPTION (0x08) Description, HISTORY_INFO_FLAG_DIFFDATA (0x10) Diff Data, HISTORY_INFO_FLAG_LASTDOCDATA 최근 문서(선택), HISTORY_INFO_FLAG_LOCK (0x40) Lock.

**option**: HWPVERSION_AUTOSAVE (0x00000001) 자동 저장.

###### 4.4.2.2. 히스토리 아이템 정보 끝

Tag ID: HISTORY_RECORD_TYPE_ETAG (0x11). **표 155**: 첨부 데이터 NONE.

###### 4.4.2.3. 히스토리 아이템 버전

Tag ID: HISTORY_RECORD_TYPE_VERSION (0x20). **표 156**: 첨부 데이터 DWORD.

###### 4.4.2.4. 히스토리 날짜

Tag ID: HISTORY_RECORD_TYPE_DATE (0x21). **표 157**: 첨부 데이터 SYSTEMDATE.

###### 4.4.2.5. 히스토리 작성자

Tag ID: HISTORY_RECORD_TYPE_WRITER (0x22). **표 158**: 첨부 데이터 WCHAR.

###### 4.4.2.6. 히스토리 설명

Tag ID: HISTORY_RECORD_TYPE_DESCRIPTION (0x23). **표 159**: 첨부 데이터 WCHAR.

###### 4.4.2.7. 비교 정보

Tag ID: HISTORY_RECORD_TYPE_DIFFDATA (0x30). **표 160**: 비교 정보(DiffML) WCHAR.

###### 4.4.2.8. 가장 마지막 최근 문서

Tag ID: HISTORY_RECORD_TYPE_LASTDOCDATA (0x31). **표 161**: 가장 마지막 최근 문서(HWPML) WCHAR.
