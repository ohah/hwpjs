**표 44: 문단 모양 속성1** — bit 0-1: 줄 간격 종류, bit 2-4: 정렬 방법(양쪽/왼쪽/오른쪽/가운데/배분/나눔), bit 5-6: 줄 나눔 기준 영문 단위, bit 7: 줄 나눔 기준 한글 단위, bit 8: 편집 용지 줄 격자 사용, bit 9-15: 최소 공백 값, bit 16: 외톨이 줄 보호, bit 17: 다음 문단과 함께, bit 18: 문단 보호, bit 19: 문단 앞에서 항상 쪽 나눔, bit 20-21: 세로 정렬, bit 22: 글꼴에 맞는 줄 높이, bit 23-24: 문단 머리 모양 종류(없음/개요/번호/글머리표), bit 25-27: 문단 수준(1~7), bit 28: 문단 테두리 연결, bit 29: 문단 여백 무시, bit 30: 문단 꼬리 모양

**표 45: 문단 모양 속성2** — bit 0-1: 한 줄 입력, bit 4: 한글과 영문 사이 자동 간격 조정, bit 5: 한글과 숫자 사이 자동 간격 조정

**표 46: 줄 간격 종류** — 한글 2007 이하: bit 0-1 (글자에 따라/고정값/여백만). 5.0.2.5 이상: bit 0-4 (글자에 따라/고정값/여백만/최소값)
