# 4.3.10.1. 구역 정의

Tag ID: MAKE_4CHID('s','e','c','d')

**표 129: 구역 정의**

| 자료형 | 길이(바이트) | 설명 |
|---|---|---|
| UINT32 | 4 | 속성(표 130) |
| HWPUNIT16 | 2 | 단 사이 간격 |
| HWPUNIT16 | 2 | 세로 줄맞춤 (0=off, 1-n=간격) |
| HWPUNIT16 | 2 | 가로 줄맞춤 (0=off, 1-n=간격) |
| HWPUNIT | 4 | 기본 팁 간격 |
| UINT16 | 2 | 번호 문단 모양 ID |
| UINT16 | 2 | 쪽 번호 (0=이어서, n=시작 번호) |
| UINT16 array[3] | 6 | 그림·표·수식 번호 |
| UINT16 | 2 | 대표 Language (5.0.1.5 이상) |
| BYTE stream | 40 | 용지설정(표 131) |
| BYTE stream | 26 | 각주 모양(표 133) |
| BYTE stream | 26 | 미주 모양(표 133) |
| BYTE stream | 12 | 쪽 테두리/배경(표 135) |

**표 130: 구역 정의 속성** — bit 0-5 머리말/꼬리말/바탕쪽/테두리/배경/쪽번호 감춤, bit 8-9 첫 쪽만 테두리/배경, bit 16-18 텍스트 방향, bit 19 빈칸 감춤, bit 20-21 페이지 번호 적용, bit 22 원고지 정서법.

**4.3.10.1.1 용지 설정** Tag: HWPTAG_PAGE_DEF. **표 131**: HWPUNIT 가로·세로·좌·우·상·하·머리말·꼬리말·제본 여백, UINT32 속성(표 132). 전체 40. **표 132**: bit 0 용지 방향, bit 1-2 제책 방법.

**4.3.10.1.2 각주/미주 모양** Tag: HWPTAG_FOOTNOTE_SHAPE. **표 133**: 속성(표 134), WCHAR 사용자/앞뒤 장식, UINT16 시작번호, 여백·구분선 등. 전체 26. **표 134**: 번호 모양, 각주 배열 방법, numbering 등.

**4.3.10.1.3 쪽 테두리/배경** Tag: HWPTAG_PAGE_BORDER_FILL. **표 135**: 속성(표 136), HWPUNIT16 4방향 간격, UINT16 테두리/배경 ID. 전체 12. **표 136**: 위치 기준, 머리말/꼬리말 포함, 채울 영역. **표 137: 바탕쪽 정보** — HWPUNIT 폭·높이, BYTE 텍스트/번호 참조. 전체 10.
