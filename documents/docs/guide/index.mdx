## 지원 기능

- ✅ **JSON 변환** (`toJson`): HWP 문서 구조를 JSON으로 추출
- ✅ **Markdown 변환** (`toMarkdown`): 텍스트, 테이블, 이미지, 하이퍼링크 지원
- ✅ **HTML 변환** (`toHtml`): CSS 스타일링 적용된 HTML 문서 생성
- ✅ **파일 헤더** (`fileHeader`): 버전, 압축/암호화 여부 빠른 조회
- ✅ **CLI**: to-json, to-markdown, to-html, info, extract-images, batch

## 플랫폼 지원

- ✅ **Web**: 완전 지원 (WASM)
- ✅ **React Native**: 완전 지원 (Craby)
