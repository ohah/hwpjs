# 7.4 OWPML의 색상 표현

OWPML 문서 내의 색상 표현은 아래 표와 같은 패턴의 형식으로 표현될 수 있다. 이 값은 기본적으로 문자열이며 HEX 숫자를 뜻하며 `#` 기호를 앞에 붙여 나타낸다.

## 표 9 -- 색상 형식 (RGBColorType)

| 항목 | 값 |
|---|---|
| **Base Type** | `xs:string` |
| **Extend Type** | HEX value |
| **Pattern** | `#[0-9A-Fa-f]{6}` |
| **Example** | `#01F39B` |
