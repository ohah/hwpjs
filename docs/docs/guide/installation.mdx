# 설치하기

HWPJS는 여러 플랫폼에서 사용할 수 있습니다. 사용할 환경에 맞는 설치 방법을 선택하세요.

## Node.js

Node.js 환경에서는 다양한 패키지 매니저를 통해 설치할 수 있습니다.

<PackageManagerTabs command="install @ohah/hwpjs" />

### Deno 사용법

Deno에서는 npm 스펙을 사용하여 설치할 수 있습니다:

```typescript
import { parseHwp } from 'npm:@ohah/hwpjs';
```

또는 `deno.json`에 의존성을 추가할 수 있습니다:

```json title="deno.json"
{
  "imports": {
    "@ohah/hwpjs": "npm:@ohah/hwpjs@^1.0.0"
  }
}
```

### 사용 예제

```typescript
import { parseHwp } from '@ohah/hwpjs';

const result = parseHwp('./document.hwp');
console.log(result);
```

## React Native

React Native 환경에서는 Craby를 통해 네이티브 모듈로 사용할 수 있습니다.

### 설치

<PackageManagerTabs command="install @ohah/hwpjs" />

### iOS 설정

iOS 프로젝트에서 CocoaPods를 사용하는 경우, `Podfile`에 다음을 추가합니다:

```ruby
pod 'ReactNative', :path => '../node_modules/@ohah/hwpjs'
```

그리고 CocoaPods 의존성을 설치합니다:

```bash
cd ios && pod install
```

### Android 설정

Android 프로젝트는 자동으로 설정됩니다. 추가 설정이 필요하지 않습니다.

### 사용 예제

```typescript
import { ReactNative } from '@ohah/hwpjs';
import RNFS from 'react-native-fs';

// 파일을 읽어서 number[]로 변환
const filePath = `${RNFS.DocumentDirectoryPath}/document.hwp`;
const fileData = await RNFS.readFile(filePath, 'base64');
const numberArray = [...Uint8Array.from(atob(fileData), (c) => c.charCodeAt(0))];

// HWP 파일 파싱
const result = ReactNative.hwp_parser(numberArray);
console.log(result);
```

더 자세한 예제는 [예제 페이지](./examples)를 참고하세요.

## 요구사항

### Node.js

- Node.js >= 12.22.0 < 13 || >= 14.17.0 < 15 || >= 15.12.0 < 16 || >= 16.0.0

### Rust (개발 시)

개발 환경에서는 Rust가 필요합니다. [rustup](https://rustup.rs/)을 통해 설치할 수 있습니다.

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
